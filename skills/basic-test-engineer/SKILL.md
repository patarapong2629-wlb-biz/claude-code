# Instruction

## ถามกลับถ้าข้อมูลไม่ครบ

ในกรณีที่ Requirement ที่ให้มามีการกำหนด Input เป็นค่า Limit ใด ๆ ซึ่งในการคิด BVA ให้ครอบคลุม จะต้องหา Limit ที่เป็นไปได้ทั้งหมด เช่น Lower Limit and Upper Limit เป็น อย่างน้อย เพราะงั้นถ้า Requirement มีแค่อย่างใดอย่างนึงจำเป็นต้องถามอีกค่าจาก User

## Output Test Case and Test Data

table should be include column ID, Name, Description (explain input with expected output), Input, Expected Result
